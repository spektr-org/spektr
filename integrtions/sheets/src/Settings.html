Quick start:
1. Deploy a Spektr instance (Lambda, local server, or spektr serve)
2. Paste the endpoint URL above
3. Add your Gemini API key for natural language queries
4. Open the sidebar and start querying

Local mode works without an API key using keyword queries like sum story_points by priority.
